# 🎤 Driver Voice Assistant - Quick Start Guide

## What's New?

The Driver Portal now has a **Voice Assistant** feature specifically designed to help drivers handle disability-related pickups with real-time position announcements.

## Where to Find It?

1. **Login as Driver** to access the Driver Dashboard
2. Look for two tabs at the top:
   - 📋 **Dashboard** - Original driver features
   - 🎤 **Voice Assistant - Disability Pickups** ← NEW TAB

## How to Use the Voice Assistant

### Step 1: Switch to Voice Assistant Tab

Click on the **"🎤 Voice Assistant - Disability Pickups"** tab to see all accessibility pickups.

### Step 2: Voice Announcement Options

You have 4 buttons to get voice announcements:

#### 🔊 **Announce All Pickups**

- Listens to all pending disability pickups
- Hears total count and details for each
- Includes:
  - Passenger name
  - Boarding location
  - Distance from depot
  - Disability type
  - Special instructions

#### 🗺️ **Announce Nearest Pickup**

- Only announces the closest passenger
- Fastest way to know your next stop
- Always picks the nearest location

#### 🎤 **Announce This Position** (Individual Pickup)

- For each pickup, you can click to hear its details
- Includes exact distance in kilometers
- Gives specific location and passenger info

#### 🗣️ **Navigation Instructions** (Individual Pickup)

- Detailed voice guidance to reach the pickup
- Distance information
- Disability-specific tips for communication

### Action Buttons

For each pickup request:

- **🎤 Announce This Position** - Hear details about this pickup
- **🗣️ Navigation Instructions** - Get driving instructions
- **✅ Acknowledge** - Confirm you've received the request
- **🏁 Complete** - Mark pickup as completed

## Disability-Specific Communication Tips

### 👁️ BLIND Passengers

✅ DO:

- Honk loudly when arriving
- Announce your arrival verbally
- Speak clearly about location
- Guide them verbally to the bus

❌ DON'T:

- Expect them to wave or signal
- Rush them during boarding
- Use only hand signals without speaking

### 🔇 DEAF Passengers

✅ DO:

- Flash headlights when arriving
- Use hand signals
- Display visual information
- Write important messages

❌ DON'T:

- Rely only on voice/honking
- Speak without looking at them
- Speak too fast or unclearly

### 🦽 MOBILITY IMPAIRMENT Patients

✅ DO:

- Park very close to boarding point
- Offer assistance if needed
- Allow extra time for boarding
- Check wheelchair accessibility

❌ DON'T:

- Park far away
- Rush them
- Assume what help they need

### 🤚 HAND DISABILITY Passengers

✅ DO:

- Park close to boarding point
- Offer help when needed
- Be patient
- Allow extra boarding time

❌ DON'T:

- Park far away
- Rush boarding
- Grab without asking

## Tips for Best Experience

1. **Enable Sound** - Make sure your device sound is on
2. **Clear Speech** - Voice commands work best in clear audio
3. **Distance Reference** - Madhavaram Depot is your reference point (0 km)
4. **Check GPS** - If distance seems wrong, ask passenger to share location again
5. **Patient Communication** - Take extra time with accessibility pickups

## Keyboard Shortcuts

## Common Questions

**Q: Why don't I see any pickups?**
A: No pending disability-related requests at this time. Check the Dashboard tab or wait for new requests.

**Q: Can I hear announcements without looking at screen?**
A: Yes! Once you click an announcement button, the voice will play. You can continue driving while listening.

**Q: How accurate is the distance?**
A: Distance is calculated from Madhavaram Depot using GPS coordinates. Accuracy is typically ±50 meters.

**Q: What if the distance shows 0?**
A: GPS might not have been shared yet. Ask passenger to share their location in the app.

**Q: Can I skip a pickup?**
A: Never skip accessibility pickups. Contact admin if there's an issue.

## Emergency Contact

If you encounter any issues with the voice assistant or the pickup:

1. Call the passenger (if phone number is available)
2. Contact admin portal
3. Use Dashboard tab to see all request details
